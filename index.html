JavaScript Variables

Variable is just a named container to store, change & update information or data.

Variables are not values. Variables point to values.

What about naming a variable -

    Valid variable names should:
  • Start with a letter, underscore, or dollar sign.
  • After the first letter, you can use numbers, letters, underscores, dollar signs.

We first declare the variable and then we assign the value

default value: undefined

  • To create a variable in JavaScript, use the var keyword.

    var allows both Re-declaration & Re-assignment of variable.

  • To create a variable in JavaScript, use the let keyword.

    let allow only Re-assignment it throws an Error when we perform Re-declaration of variable.

  • To create a variable in JavaScript, use the const keyword.

    const doesn't allow both Re-declaration & Re-assignment of variable.
